# **App Name**: Animatrix

## Core Features:

- Video Browsing: Browse and stream a curated collection of animation clips.
- Hover Play: On-hover play functionality for quick previewing of animation clips.
- Save Videos: Save animation clips to a personal collection.
- Share Videos: Share favorite clips via social media.
- Like Videos: Like videos.
- AI Recommendations: Use an AI tool that watches your view history and makes video clip recommendations

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to capture a sense of depth and creativity, aligning with the artistry of animation.
- Background color: Dark Gray (#222222) to create a cinematic viewing experience, similar to Netflix.
- Accent color: Electric Purple (#8F00FF) to highlight interactive elements and calls to action, ensuring a vibrant user experience.
- Body and headline font: 'Inter', a sans-serif font, to offer a modern, clean, and easily readable interface for browsing animation clips.
- Use clean and simple icons for actions like play, save, share, and like.
- Netflix-inspired grid layout for displaying animation clips, focusing on visual presentation and easy navigation.
- Smooth transition animations when hovering over clips or navigating between pages, enhancing the user experience with subtle feedback.
